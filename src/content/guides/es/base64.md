Base64 es un esquema de codificación de binario a texto que representa datos binarios en formato de cadena ASCII. Se utiliza comúnmente para incrustar imágenes en HTML, transmitir datos en APIs y almacenar datos complejos en formatos de texto.

### Cómo Usarlo

1. **Codificar**: Ingrese texto normal en el campo de entrada y la versión codificada en Base64 aparece automáticamente.
2. **Decodificar**: Pegue texto codificado en Base64 y el texto original decodificado aparece al instante.
3. **Copiar**: Use el botón de copiar para obtener el resultado codificado o decodificado.

### Usos Comunes

- Incrustar imágenes como data URIs en HTML y CSS
- Codificar datos binarios para solicitudes de API
- Almacenar datos complejos en cadenas JSON

### Privacidad

Todas las conversiones ocurren en su navegador. Ningún dato se envía a ningún servidor.
