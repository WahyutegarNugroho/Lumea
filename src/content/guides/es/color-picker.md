Elegir los colores correctos es crucial para el diseño. Lumea Color Picker le ayuda a encontrar y convertir colores para sus proyectos.

### Cómo Usarlo

1. **Seleccione un color**: Use el selector de color para elegir cualquier color.
2. **Vea los valores**: Observe el color representado en HEX, RGB y otros formatos.
3. **Copiar**: Haga clic en cualquier formato para copiar el valor a su portapapeles.
4. **Historial**: Los colores seleccionados recientemente se guardan para acceso rápido.

### Características

- Soporte de múltiples formatos de color (HEX, RGB, HSL)
- Historial de colores para reutilización rápida
- Vista previa en tiempo real de los colores seleccionados

### Privacidad

Todo se ejecuta localmente. Ningún dato de color se envía jamás a ningún servidor.
