El código limpio y bien formateado es más fácil de leer, depurar y mantener. Lumea Code Formatter utiliza Prettier para embellecer su código fuente al instante.

### Cómo Usarlo

1. **Pegue su código**: Pegue código fuente sin formato o minimizado en el editor.
2. **Elija el lenguaje**: Seleccione JavaScript, HTML o CSS para aplicar las reglas de formato correctas.
3. **Embellecer**: Haga clic en "Embellecer Código" para transformar su código en un formato limpio y legible.
4. **Copiar**: Use el botón de copiar para obtener el resultado formateado.

### Privacidad

Su código nunca sale de su computadora. Prettier se ejecuta directamente en su navegador para formatear su código fuente localmente.
