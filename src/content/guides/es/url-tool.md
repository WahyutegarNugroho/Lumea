La codificación de URL, también conocida como codificación porcentual, convierte los caracteres especiales en las URLs a un formato que puede transmitirse de forma segura a través de internet.

### Cómo Usar

1. **Para codificar**: Escriba o pegue una URL con caracteres especiales (espacios, símbolos, etc.) y la versión codificada aparecerá.
2. **Para decodificar**: Pegue una URL codificada para ver la versión legible.
3. **Copiar**: Tome el resultado para usarlo en sus aplicaciones.

### Casos de Uso Comunes

- Preparación de parámetros de consulta para llamadas API
- Manejo de URLs de redireccionamiento complejas
- Codificación de caracteres no ASCII para uso web

### Privacidad

Toda la codificación y decodificación de URL se realiza localmente en su navegador. Sus datos nunca abandonan su dispositivo.
