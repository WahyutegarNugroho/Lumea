Recortar es una de las tareas de edición de fotos más comunes. Ya sea que esté preparando imágenes para redes sociales, impresión o publicación web, Lumea lo hace fácil.

### Cómo Usarlo

1. **Sube tu imagen**: Seleccione la foto que desea recortar.
2. **Ajuste el área de recorte**: Arrastre los controladores para seleccionar la porción que desea conservar.
3. **Elija un preset**: Use proporciones estándar como 1:1 para fotos de perfil, 4:3 para presentaciones o 16:9 para videos.
4. **Recortar y descargar**: Guarde su imagen recortada al instante.

### Consejos Profesionales

- Use la proporción 1:1 para fotos de perfil perfectas.
- Para impresión, mantenga la relación de aspecto original para evitar estiramientos.
- Recorte primero, luego redimensione para obtener resultados óptimos.

### Privacidad

Todo el recorte ocurre localmente en su navegador. Sus fotos nunca salen de su dispositivo.
