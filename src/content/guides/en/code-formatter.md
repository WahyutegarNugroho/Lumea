Clean, well-formatted code is easier to read, debug, and maintain. Lumea Code Formatter uses Prettier to beautify your source code instantly.

### How to Use

1. **Paste your code**: Paste unformatted or minified source code into the editor.
2. **Choose language**: Select JavaScript, HTML, or CSS to apply correct formatting rules.
3. **Beautify**: Click "Beautify Code" to transform your code into a clean, readable format.
4. **Copy**: Use the copy button to grab the formatted result.

### Privacy

Your code never leaves your computer. Prettier runs directly in your browser to format your source code locally.
