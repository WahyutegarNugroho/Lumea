Choosing the right colors is crucial for design. Lumea Color Picker helps you find and convert colors for your projects.

### How to Use

1. **Pick a color**: Use the color picker input to select any color.
2. **View values**: See the color represented in HEX, RGB, and other formats.
3. **Copy**: Click any format to copy the value to your clipboard.
4. **History**: Recently picked colors are saved for quick access.

### Features

- Multiple color format support (HEX, RGB, HSL)
- Color history for quick reuse
- Real-time preview of selected colors

### Privacy

Everything runs locally. No color data is ever sent to any server.
