JSON is the backbone of modern web APIs. Lumea JSON Formatter helps developers clean up messy JSON data and spot syntax errors.

### Features

- **Beautify**: Adds proper indentation and newlines for readability.
- **Minify**: Removes all whitespace to reduce file size.
- **Validation**: Shows exactly where your JSON structure is broken.

### How to Use

1. Paste your JSON data into the input area.
2. Click "Beautify" to format with proper indentation.
3. Click "Minify" to compress the JSON.
4. Copy the result for use in your code.

### Privacy

Your JSON data is processed entirely in your browser. No data is ever sent to any server.
