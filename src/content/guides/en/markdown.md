Lumea Markdown Editor provides a seamless writing experience with instant visualization and professional export options.

### Key Features

- **Real-time Preview**: See your formatted text immediately as you type with a side-by-side view.
- **HTML Export**: Convert your Markdown to clean HTML code with one click.
- **Download Markdown**: Save your work as a .md file for later use.

### How to Use

1. Write or paste Markdown text in the editor panel on the left.
2. Watch the formatted preview update in real time on the right.
3. Use the toolbar buttons to copy HTML or download your markdown file.

### Privacy

Everything stays on your machine. Your text is never sent to any server, protecting your content and intellectual property.
