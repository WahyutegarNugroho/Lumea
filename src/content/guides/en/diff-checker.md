Comparing text changes is essential for code reviews, document editing, and version tracking. Lumea Diff Checker uses an efficient LCS algorithm.

### How to Use

1. **Paste original**: Enter the original text in the left panel.
2. **Paste modified**: Enter the changed text in the right panel.
3. **Compare**: Click "Compare Now" to see the differences highlighted.
4. **Review**: Additions, deletions, and changes are clearly marked.

### Features

- Zero-dependency engine using LCS algorithm
- Side-by-side comparison view
- Instant results without server round-trips

### Privacy

The comparison runs entirely in your browser. Your text is never sent to any server.
