Base64 is a binary-to-text encoding scheme that represents binary data in an ASCII string format. It is commonly used for embedding images in HTML, transmitting data in APIs, and storing complex data in text formats.

### How to Use

1. **Encode**: Enter normal text in the input field, and the Base64 encoded version appears automatically.
2. **Decode**: Paste Base64 encoded text, and the decoded original text appears instantly.
3. **Copy**: Use the copy button to grab either the encoded or decoded result.

### Common Uses

- Embedding images as data URIs in HTML and CSS
- Encoding binary data for API requests
- Storing complex data in JSON strings

### Privacy

All conversions happen in your browser. No data is sent to any server.
