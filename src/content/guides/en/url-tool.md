URL encoding, also known as percent-encoding, converts special characters in URLs to a format that can be safely transmitted over the internet.

### How to Use

1. **To encode**: Type or paste a URL with special characters (spaces, symbols, etc.) and the encoded version appears.
2. **To decode**: Paste an encoded URL to see the human-readable version.
3. **Copy**: Grab the result for use in your applications.

### Common Use Cases

- Preparing query parameters for API calls
- Handling complex redirect URLs
- Encoding non-ASCII characters for web use

### Privacy

All URL encoding and decoding is performed locally in your browser. Your data never leaves your device.
