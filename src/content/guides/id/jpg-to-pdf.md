Mengonversi gambar ke PDF adalah tugas umum untuk membuat portofolio, mendigitalkan kwitansi, atau menyusun album foto. Lumea membuatnya sederhana.

### Langkah demi Langkah

1. **Unggah gambar Anda**: Seret dan lepas file JPG, PNG, atau gambar lainnya ke dalam alat. Anda dapat menambahkan beberapa gambar sekaligus.
2. **Atur urutan**: Gambar akan muncul dalam daftar. Anda dapat menghapus dan menambahkannya kembali untuk menyesuaikan urutan.
3. **Konversi**: Klik "Convert to PDF" untuk menggabungkan semua gambar menjadi satu dokumen PDF.
4. **Unduh**: Browser Anda akan menyimpan PDF baru secara otomatis.

### Optimasi Otomatis

Gambar secara otomatis diukur agar sesuai dengan dimensi halaman standar. PDF yang dihasilkan dioptimalkan untuk pencetakan dan distribusi digital.

### Privasi

Gambar Anda diproses sepenuhnya di browser Anda. Gambar tidak pernah diunggah ke server mana pun, memastikan konten visual Anda tetap pribadi.
