Base64 adalah skema encoding biner-ke-teks yang merepresentasikan data biner dalam format string ASCII. Ini umum digunakan untuk menyematkan gambar di HTML, mentransmisikan data di API, dan menyimpan data kompleks dalam format teks.

### Cara Menggunakan

1. **Encode**: Masukkan teks biasa di kolom input, dan versi terenkode Base64 muncul secara otomatis.
2. **Decode**: Tempel teks terenkode Base64, dan teks asli yang didekode muncul secara instan.
3. **Salin**: Gunakan tombol salin untuk mengambil hasil encode atau decode.

### Penggunaan Umum

- Menyematkan gambar sebagai data URI di HTML dan CSS
- Meng-encode data biner untuk permintaan API
- Menyimpan data kompleks dalam string JSON

### Privasi

Semua konversi terjadi di browser Anda. Tidak ada data yang dikirim ke server mana pun.
