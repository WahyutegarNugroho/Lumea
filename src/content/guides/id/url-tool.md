Enkode URL, juga dikenal sebagai persen-enkode, mengonversi karakter khusus dalam URL ke format yang dapat ditransmisikan dengan aman melalui internet.

### Cara Menggunakan

1. **Untuk mengenkode**: Ketik atau tempel URL dengan karakter khusus (spasi, simbol, dll.) dan versi terenkode akan muncul.
2. **Untuk mendekode**: Tempel URL yang telah dienkode untuk melihat versi yang dapat dibaca manusia.
3. **Salin**: Ambil hasilnya untuk digunakan dalam aplikasi Anda.

### Kasus Penggunaan Umum

- Menyiapkan parameter kueri untuk panggilan API
- Menangani URL pengalihan yang kompleks
- Mengenkode karakter non-ASCII untuk penggunaan web

### Privasi

Semua enkode dan dekode URL dilakukan secara lokal di peramban Anda. Data Anda tidak pernah meninggalkan perangkat Anda.
