Gambar vektor didefinisikan secara matematis, artinya dapat diskalakan ke ukuran apa pun tanpa menjadi pecah. Ini penting untuk logo, ikon, dan ilustrasi.

### Mengapa Vektorisasi?

- **Skalabilitas tak terbatas**: SVG terlihat tajam dalam ukuran apa pun, dari favicon hingga papan reklame.
- **Ukuran file kecil**: Jalur vektor lebih efisien daripada data piksel.
- **Kemudahan pengeditan**: SVG dapat diedit di perangkat lunak desain vektor apa pun.

### Cara Menggunakan

1. **Unggah gambar Anda**: Pilih gambar raster (JPG, PNG) untuk dikonversi.
2. **Pilih prasetel**: Pilih dari "Default", "Logo / Datar" untuk tepi tajam, "Detail Tinggi" untuk gambar kompleks, atau "Hitam & Putih".
3. **Vektorisasi**: Klik untuk memproses gambar dan menghasilkan jalur SVG.
4. **Unduh**: Simpan file SVG yang dihasilkan.

### Privasi

Gambar Anda diproses sepenuhnya di peramban Anda. Tidak ada data yang dikirim ke server eksternal, memastikan desain Anda tetap privat.
