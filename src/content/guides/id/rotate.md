Pernahkah Anda membuka PDF hanya untuk menemukan bahwa beberapa halaman diputar secara tidak benar? Alat Putar Lumea memungkinkan Anda memperbaikinya secara permanen.

### Cara Menggunakan

1. **Unggah PDF Anda**: Pilih file yang ingin Anda perbaiki.
2. **Pilih rotasi**: Klik "90° Kiri" atau "90° Kanan" untuk memutar semua halaman dalam dokumen.
3. **Pratinjau**: Alat memproses rotasi dan memperbarui dokumen.
4. **Unduh**: Klik "Simpan & Unduh PDF" untuk mendapatkan file yang telah diperbaiki.

### Perbaikan Permanen

Rotasi diterapkan langsung ke metadata halaman internal PDF. Ini berarti orientasi tetap terkunci tidak peduli pembaca PDF atau perangkat apa yang Anda gunakan untuk membuka file.

### Privasi Utama

Dokumen Anda diproses sepenuhnya di dalam peramban Anda. File tidak pernah meninggalkan komputer Anda, memastikan privasi lengkap.
