Memilih warna yang tepat sangat penting untuk desain. Lumea Color Picker membantu Anda menemukan dan mengonversi warna untuk proyek Anda.

### Cara Menggunakan

1. **Pilih warna**: Gunakan input pemilih warna untuk memilih warna apa pun.
2. **Lihat nilai**: Lihat warna yang direpresentasikan dalam HEX, RGB, dan format lainnya.
3. **Salin**: Klik format apa pun untuk menyalin nilai ke clipboard Anda.
4. **Riwayat**: Warna yang baru dipilih disimpan untuk akses cepat.

### Fitur

- Dukungan berbagai format warna (HEX, RGB, HSL)
- Riwayat warna untuk penggunaan kembali yang cepat
- Pratinjau real-time dari warna yang dipilih

### Privasi

Semuanya berjalan secara lokal. Tidak ada data warna yang pernah dikirim ke server mana pun.
