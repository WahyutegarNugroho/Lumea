JSON adalah tulang punggung API web modern. Lumea JSON Formatter membantu pengembang membersihkan data JSON yang berantakan dan menemukan kesalahan sintaks.

### Fitur

- **Perindah**: Menambahkan indentasi dan baris baru yang tepat untuk keterbacaan.
- **Minifikasi**: Menghapus semua spasi untuk mengurangi ukuran file.
- **Validasi**: Menunjukkan dengan tepat di mana struktur JSON Anda rusak.

### Cara Menggunakan

1. Tempel data JSON Anda ke area input.
2. Klik "Beautify" untuk memformat dengan indentasi yang tepat.
3. Klik "Minify" untuk mengompresi JSON.
4. Salin hasilnya untuk digunakan dalam kode Anda.

### Privasi

Data JSON Anda diproses sepenuhnya di browser Anda. Tidak ada data yang pernah dikirim ke server mana pun.
