Lumea Markdown Editor menyediakan pengalaman menulis yang mulus dengan visualisasi instan dan opsi ekspor profesional.

### Fitur Utama

- **Pratinjau Real-time**: Lihat teks yang diformat segera saat Anda mengetik dengan tampilan berdampingan.
- **Ekspor HTML**: Konversi Markdown Anda ke kode HTML yang bersih dengan satu klik.
- **Unduh Markdown**: Simpan pekerjaan Anda sebagai file .md untuk digunakan nanti.

### Cara Menggunakan

1. Tulis atau tempel teks Markdown di panel editor di sebelah kiri.
2. Saksikan pratinjau yang diformat diperbarui secara real-time di sebelah kanan.
3. Gunakan tombol toolbar untuk menyalin HTML atau mengunduh file markdown Anda.

### Privasi

Semuanya tetap di mesin Anda. Teks Anda tidak pernah dikirim ke server mana pun, melindungi konten dan kekayaan intelektual Anda.
