Membandingkan perubahan teks sangat penting untuk review kode, pengeditan dokumen, dan pelacakan versi. Lumea Diff Checker menggunakan algoritma LCS yang efisien.

### Cara Menggunakan

1. **Tempel asli**: Masukkan teks asli di panel kiri.
2. **Tempel modifikasi**: Masukkan teks yang diubah di panel kanan.
3. **Bandingkan**: Klik "Compare Now" untuk melihat perbedaan yang disorot.
4. **Tinjau**: Penambahan, penghapusan, dan perubahan ditandai dengan jelas.

### Fitur

- Engine tanpa dependensi menggunakan algoritma LCS
- Tampilan perbandingan berdampingan
- Hasil instan tanpa perjalanan bolak-balik ke server

### Privasi

Perbandingan berjalan sepenuhnya di browser Anda. Teks Anda tidak pernah dikirim ke server mana pun.
