Kode yang bersih dan terformat dengan baik lebih mudah dibaca, di-debug, dan dipelihara. Lumea Code Formatter menggunakan Prettier untuk memperindah kode sumber Anda secara instan.

### Cara Menggunakan

1. **Tempel kode Anda**: Tempel kode sumber yang tidak terformat atau diminifikasi ke dalam editor.
2. **Pilih bahasa**: Pilih JavaScript, HTML, atau CSS untuk menerapkan aturan format yang benar.
3. **Perindah**: Klik "Beautify Code" untuk mengubah kode Anda menjadi format yang bersih dan mudah dibaca.
4. **Salin**: Gunakan tombol salin untuk mengambil hasil yang telah diformat.

### Privasi

Kode Anda tidak pernah meninggalkan komputer Anda. Prettier berjalan langsung di browser Anda untuk memformat kode sumber Anda secara lokal.
